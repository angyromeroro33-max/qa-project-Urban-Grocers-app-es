## Objetivo
Verificar que la API gestione correctamente la creación de kits de
productos, validando campos como name y comprobando que el sistema
responda adecuadamente ante datos válidos e inválidos.

## Qué se valida
- Creación correcta de kits de productos
- Validación del campo name
- Respuestas de error cuando los datos no cumplen las reglas
- Comportamiento esperado de la API en distintos escenarios

## Tecnologías utilizadas
- Python
- Requests
- Automatización de pruebas de API

## Propósito del proyecto
Garantizar la calidad del sistema mediante pruebas automatizadas,
detectando errores de forma temprana y asegurando el correcto
funcionamiento de la aplicación Urban Grocers.
